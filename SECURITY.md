# Security Policy

## Supported Versions

This is an educational project. Only the latest version on `master` is actively maintained.

| Version | Supported |
|---------|-----------|
| latest (master) | Yes |
| older tags | No |

## Reporting a Vulnerability

This project does not handle user credentials, network connections, or sensitive data. It is a local numerical computing library.

If you discover a dependency with a known CVE (e.g. in NumPy, SciPy, or Streamlit), please open a regular GitHub issue titled `[SECURITY] <package> CVE-XXXX-XXXXX` and we will update the dependency pin promptly.

For any other security concern, open a GitHub issue with the `security` label.
